# 💡 生きものレッドリスト横断検索サービス — マネタイズ構想 v2

## 基本思想

> **「環境調査のプロが毎回やっている、最も不毛な作業——複数自治体のレッドリストとのリスト突合——をゼロにする」**

一般メディア型（PV×広告単価）ではなく、**専門家の時間を売るBtoB SaaSモデル**を軸に据える。ニッチさは弱点ではなく、ターゲット精度の高さとして武器になる。

---

## 収益柱の整理

### 柱①　フリーミアム（サブスクリプション）— メイン収益

| 無料プラン | 有料プラン（想定：月3,000〜8,000円） |
|-----------|--------------------------------------|
| 種名・カテゴリ・都道府県でのキーワード検索 | **自社リストとの一括照合**（Excel/CSV 1,000件単位） |
| 1件ずつの手動確認 | **最新データのCSV一括エクスポート** |
| 薄いAdSense広告（サーバー代程度） | **APIアクセス**（社内システム・GISとの連携） |
| — | **更新通知**（指定種・カテゴリ変更をメール通知） |
| — | 広告非表示 |

**想定顧客**：環境コンサルタント会社（社員単位 or 事務所単位）、大学研究室、NPO

**競合との差別化ポイント**：既存の環境データベース（自然環境情報WebGIS等）は「検索できる」が「自分のリストと突合できる」ものがない。

---

### 柱②　専門特化型の純広告・求人枠 — 安定収益

AdSenseのように第三者が入札する形式ではなく、**関連業界へ直販**する。ターゲット一致率が高いため、CPMではなく固定月額で交渉できる。

**広告主候補**：
- 環境アセスメント会社の**求人バナー**（中途採用、野外調査員）
- 野外調査機材メーカー（GPS、双眼鏡、ドローン、サンプリング器具）
- 環境系資格予備校・通信教育
- 環境系学会・研究会のイベント告知

**価格感の根拠**：月間UU 5,000人でも、全員が職業的な自然環境関係者であれば広告主にとってのCPMは一般メディアの数倍〜数十倍の価値がある。

---

### 柱③　GIS統合ツール — 将来の高付加価値化

現在実装済みの都道府県マップを起点に、中長期で発展させる方向性。

**ロードマップ案**：

1. **現在**：都道府県ごとの指定状況マップ（実装済み）
2. **次段階**：環境省メッシュデータ（第3次自然環境保全基礎調査）との重ね合わせ
3. **将来**：任意地点の緯度経度を入力 → 周辺X km圏内の絶滅危惧種一覧を出力

フェーズ3のGIS化が実現すると、環境アセスメントの**現地調査前スクリーニング**をほぼ自動化できる。これは現状、調査員が複数のPDFを手で突合している工程にあたり、**最もビリング可能な価値**になりうる。

## 実装優先度と収益化のタイミング

```
現在（フェーズ2完了）
  → データ拡充（47都道府県化）が最優先
  → 薄いAdSense導入（サーバー代の補填）
        ↓
データが30都道府県超えたら
  → 有料機能のベータ提供開始（一括照合）
  → 環境系求人サイト・機材メーカーへの広告枠交渉
        ↓
47都道府県＋主要市区町村対応後
  → 正式サブスク開始
  → GIS連携の検討（Supabase移行タイミングと重なる）
```

## 注意事項・リスク

**ライセンス**：現在使用中のGeolonia SVG（GFDL）は商用化時に問題になりうる。収益化前に自作SVGまたはライセンスクリアな地図データへの切り替えが必要。

**データの権利**：各自治体のレッドリストデータの二次利用条件を確認しておく必要がある。公的データであっても商用サービスでのAPI提供時には確認が求められる場合がある。

**競合参入**：データ整備が進むにつれ、大手環境データ会社が類似機能を追加するリスクはある。先行者利益と個人開発者ならではのコスト優位を活かして早期に認知を獲得することが重要。
